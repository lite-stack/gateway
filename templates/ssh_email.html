New SSH keys were created

A new pair of SSH keys were created for connecting for server. This pair will be used for all your servers. There are steps how to use these keys:

  • Add the keys to your keys directory:
    • For Windows: copy to C:\Users\<your_user>/.ssh/
    • For Linux: copy to ~/.ssh/
  • Go to the directory with SSH keys:
    • For Windows: cd C:\Users\<your_user>/.ssh/
    • For Linux: cd ~/.ssh/
      chmod 600 dev_stack.pem
  • Connect to the server using ssh command:
    ssh -i devstack root@{{public_address}}
